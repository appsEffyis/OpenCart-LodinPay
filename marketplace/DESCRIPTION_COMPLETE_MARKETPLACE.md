## 📖 Easy Installation

### Automatic Installation (Recommended)
1. Download the extension ZIP file
2. Go to **Extensions → Installer** in OpenCart admin
3. Upload the ZIP file
4. Go to **Extensions → Extensions → Payments**
5. Find **LodinPay** and click Install
6. Click Edit to configure

### Manual Installation
Detailed instructions included in the PDF documentation.

---

## ⚙️ Simple Configuration

1. **Get Your Credentials**
   - Sign up at https://lodinpay.com
   - Get your Client ID and Client Secret

2. **Configure Extension**
   - Enter Client ID
   - Enter Client Secret
   - Select Environment (Sandbox/Production)
   - Set order statuses
   - Enable the payment method

3. **Setup Webhook**
   - Copy webhook URL from settings
   - Add to LodinPay merchant dashboard
   - Test webhook connection

4. **Start Accepting Payments**
   - Make a test order
   - Verify payment processing
   - Go live!

## 📂 COMMENT UPLOADER UN PDF SUR LE MARKETPLACE

### Étape 1: Préparer ton PDF

1. **Crée un PDF d'installation** (10-20 pages)
   - Utilise Word, Google Docs, ou Canva
   - Inclus: Installation, Configuration, Usage
   - Ajoute des screenshots
   - Exporte en PDF

2. **Nomme le fichier**:
   ```
   LodinPay-Installation-Guide-v1.0.pdf
   ```

3. **Taille recommandée**: < 5 MB

### Étape 2: Sur le Marketplace OpenCart

1. **Connecte-toi** au marketplace:
   ```
   https://www.opencart.com/index.php?route=marketplace/seller
   ```

2. **Va dans "My Extensions"**

3. **Clique sur ton extension** (ou "Add Extension")

4. **Section "Files"** ou "Documentation":
   - Cherche le champ "Documentation" ou "Additional Files"
   - Clique sur "Upload" ou "Browse"
   - Sélectionne ton PDF
   - Upload

5. **Section "Description"**:
   - Mentionne que la documentation PDF est incluse
   - Exemple: "Complete installation guide (PDF) included"

### Étape 3: Alternative - Inclure dans le ZIP

Si le marketplace ne permet pas d'upload PDF séparé:

1. **Ajoute le PDF dans ton package ZIP**:
   ```
   lodin-opencart-1.0.0.zip
   ├── lodin/
   │   └── (fichiers extension)
   └── docs/
       ├── Installation-Guide.pdf
       ├── User-Manual.pdf
       └── README.md
   ```

2. **Mentionne dans la description**:
   ```
   📚 Complete documentation included in the package:
   - Installation Guide (PDF)
   - User Manual (PDF)
   - README file
   ```
