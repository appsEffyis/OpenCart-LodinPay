# Création des Images pour Marketplace

## Images Requises

### 1. Image (Thumbnail) - 260 x 152 px
- Utilisée sur la page marketplace
- Doit contenir le logo Lodin
- Format: PNG ou JPG
- Fond: Blanc ou transparent

### 2. Banner - 710 x 380 px
- Utilisée sur la page d'info de l'extension
- Peut inclure:
  - Logo Lodin
  - Texte: "Lodin RTP Payment"
  - Slogan: "Instant Payment Gateway"
- Format: PNG ou JPG

## Outils pour Créer les Images

### Option A: Canva (Gratuit)
1. Allez sur: https://www.canva.com/
2. Créez un design personnalisé:
   - 260 x 152 px pour thumbnail
   - 710 x 380 px pour banner
3. Ajoutez le logo Lodin
4. Téléchargez en PNG

### Option B: Photoshop / GIMP
1. Créez un nouveau document avec les dimensions
2. Ajoutez le logo Lodin
3. Exportez en PNG

### Option C: Figma (Gratuit)
1. Allez sur: https://www.figma.com/
2. Créez un frame avec les dimensions
3. Ajoutez le logo
4. Exportez en PNG

## Recommandations Design

**Thumbnail (260x152):**
- Logo Lodin centré
- Fond blanc ou bleu clair
- Texte minimal

**Banner (710x380):**
- Logo Lodin à gauche
- Texte "Lodin RTP Payment" en grand
- Sous-titre: "Instant Payment Gateway for OpenCart"
- Icônes: paiement, sécurité, rapidité
- Couleurs: Bleu et blanc (couleurs Lodin)

## Exemple de Contenu

**Banner Text:**
```
LODIN RTP PAYMENT
Instant Payment Gateway

✓ Real-time payments
✓ Secure transactions
✓ Easy integration
```

## Après Création

1. Sauvegardez les images:
   - `lodin-thumbnail-260x152.png`
   - `lodin-banner-710x380.png`

2. Uploadez sur le marketplace OpenCart

3. Ajoutez aussi des screenshots:
   - Configuration admin
   - Checkout page
   - Payment success
